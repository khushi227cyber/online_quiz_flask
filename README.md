# Online Quiz Application (Flask)

This is a simple Online Quiz Application developed using Python Flask.
The project allows users to attempt multiple choice questions and view their score.
## Features
- Start quiz
- Answer MCQ questions
- Submit quiz
- View final score
## Technologies Used
- Python (Flask)
- HTML
- CSS
- CSV file for questions
- Git & GitHub
-  ## Project Structure
The project contains Flask backend, HTML templates for frontend, CSS for styling, and a CSV file for quiz questions.
